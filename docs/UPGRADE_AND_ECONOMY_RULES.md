# Upgrade And Economy Rules

This document keeps competitive balance separate from long-term progression.

## Duel

- Duel is the core skill-based normalized mode.
- Permanent stat upgrades should not affect Duel by default.
- Duel may allow cosmetics and carefully designed sidegrades later.
- Duel rewards can grant Bolts/Crystals, but power should not snowball unfairly.

## BattleArena

- BattleArena can allow more chaos and progression than Duel.
- Temporary run upgrades are acceptable here.
- Arena progression should be fun, readable, and reversible.

## Future Extraction

- Extraction is run-based.
- Death loses unbanked loot from that run.
- Successful extraction saves loot and rewards.
- Progression can be stronger here because it is not pure competitive Duel.

## Currencies

Bolts:

- common currency;
- arena run upgrades;
- basic unlocks;
- non-competitive progression.

Crystals:

- rare unlocks;
- skins;
- VFX;
- special weapons;
- sidegrades;
- premium-feeling cosmetics.

Crystals should not become direct Duel stat advantage by default.

## Future Upgrade Types

Allowed outside normalized Duel:

- HP;
- Damage;
- Reload;
- Speed;
- Projectile Speed.

Inside normalized Duel, these values should come from the Duel ruleset unless explicitly testing a separate mode.
