# Список ассетов для добавления (Ricochet Tanks)

Всё ниже — слоты, которые УЖЕ подключены в коде, но ждут реального ассета. Пока пусто:
звук молчит (без ошибок), VFX рисует процедурный пафф, иконка показывает символ. После
импорта впиши `rbxassetid` / положи объект в нужную папку.

Приоритет: ⭐ важно для ощущения игры · ◻ желательно · ▫ опционально.

---

## 1. ЗВУКИ — `AudioCatalog.luau` (вписать `SoundId = "rbxassetid://..."`)

УЖЕ ЕСТЬ (не трогать): Shot, Ricochet, ProjectileImpact, TankDestroyed(взрыв).

НУЖНО добавить (пусто):
- ⭐ **ShotRocket** — выстрел ракетницы
- ⭐ **ShotMachineGun** — выстрел пулемёта (короткий, для очереди)
- ⭐ **ShotRailGun** — выстрел рельсы (мощный, с «зарядом»)
- ⭐ **PickupCollect** — подбор пикапа/оружия
- ⭐ **Repair** — звук починки
- ◻ **ReflectShield** — отражение щитом
- ◻ **LevelUp** — повышение уровня арены
- ◻ **UpgradeSelect** — выбор апгрейда
- ◻ **Win / Lose** — конец дуэли
- ▫ **NoPenetration** — «не пробил» (рикошет/звяк)
- ▫ **BarrelBlocked** — ствол упёрся
- ▫ **BoltReward** — начисление болтов
- ▫ **ButtonClick** — клик в UI
- ▫ **EngineLoop** — гул мотора (зацикленный)

> Для per-weapon выстрелов: если оставить пустым — играет обычный Shot (фолбэк уже есть).

---

## 2. VFX (эффекты) — папка `ReplicatedStorage/Shared/Assets/VFX`

УЖЕ ЕСТЬ: muzzle, trail, impact, ricochet, взрыв танка, level-up, reflect-shield,
trail танка (TankTrailVFX).

НУЖНО добавить объект в папку (имя важно — реестр уже на него ссылается). Пока пусто —
играет процедурный пафф нужного цвета:
- ◻ **Repair_Default** — починка (зелёные искры/плюсики)
- ◻ **UpgradeSelect_Default** — выбор апгрейда
- ◻ **RocketShot_Default** — вспышка выстрела ракетницы (оранжевая)
- ◻ **RocketExplosion_Default** — взрыв ракеты при детонации (отдельный от смерти танка)
- ◻ **RailGunShot_Default** — вспышка выстрела рельсы (синяя)

> Уникальные слоты под ракету и рельсу ДОБАВЛЕНЫ в реестр и подключены в коде:
> при выстреле Rocket/RailGun играется их muzzle-FX, при взрыве ракеты — RocketExplosion.
> Снаряд рельсы остаётся синим (цвет в коде), это про вспышку/взрыв.

После импорта: в `VfxRegistry.luau` у соответствующего эффекта убрать `Missing = true`.

## 3. ИКОНКИ АПГРЕЙДОВ — `UpgradeIconConfig.luau` (вписать `Image`)

УЖЕ ЕСТЬ PNG: DamageUp, FireRateUp, MoveSpeedUp, DoubleShot, TripleSpread, RicochetUp, Repair.

НУЖНО нарисовать + импортировать (сейчас текстовый символ):
- ◻ **Lifesteal** (♥) — хил за килл
- ◻ **Piercing** (➹) — пробитие

> Без иконки HUD показывает символ — играбельно, но не красиво.

- Соц: kill-feed, имена у HP-баров, ачивки, глобальные подиумы.
- Магазин скинов (Bolts) с превью.
- Дым танка из 2 гусениц, отражающийся луч прицела.
